== Description ==

Event Mangement Plugin For WordPress, Create Events, Manage Event Bookings, Manage Event Tickets, Event Sponsors, Beautiful And Well Designed, With Easy User Interface, Efficient For Event Booking And Ticketing

If you are looking for any free event management plugin for WordPress website or blog than Event Monster is the right choice for you. Event Monster is the simplest easy elegant powerful WordPress plugin for organizing events online. 
Accept event booking using the registration form, manage attendees, manage free & external bookings.

Event Monster has a beautiful & user-friendly interface. Very easy to use, you can create multiple events in a few minutes. Event Monster is a fully responsive design layout that means it is compatible with all types of devices like large desktop, small desktop, laptops, tablets, mobile phones.
Event Monster has the best and beautiful email template for showing booking status, details and notification of your event. Share event details with social media icons. Accept event booking and registration, manage attendees in event monster.

== Installation ==

Install event monster either via the WordPress.org plugin directory or by uploading the files to your server.

After activating event monster plugin, go to plugin menu.

Create events, configure settings and save.

Copy event monster shortcode and publish on page/post.

That's it. You're ready to go!
